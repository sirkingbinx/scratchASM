ScratchASM (ssm) is an Assembly executor and VM built in Scratch. It aims to be fast enough to run on standalone Scratch (so without TurboWarp, Penguin, etc.)

> [!NOTE]
> This project is going under a large rework, so a lot of information may be missing.
